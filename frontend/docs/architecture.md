# Architecture

These document explain how the main points of architecture works on these project
separated by section.

## 1.0 Tech Stack

Frontend tech stack uses Vite, React, Typescript and GraphQL Client.

## 2.0 Styling

For styling we always use TailwindCSS as CSS framework.

## 3.0 Component library

For component library we use ShadCN UI, all base components are located at `src/components/ui/button.tsx`

## 4.0 Testing

For testing we use vitest.

## 5.0 Data Fetching

For data fetching we use Apollo Client with useQuery and useMutation hooks.
